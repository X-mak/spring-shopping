新增售后申请单

请求URL

/requisition/ POST

请求体 application/json

{
	"id":"int //主键，售后申请编号",
	"orderItemId":"int //订单项编号",
	"topic":"string //标题",
	"introduction":"string //介绍",
	"picture":"string //图片",
	"status":"int //申请状态",
	"goods":{
		"id":"int //主键，商品编号",
		"goodsName":"string //商品名",
		"price":"double //商品价格",
		"stock":"int //商品库存",
		"goodsStatus":"int //商品状态，1为销售中，0为已下架",
		"introduction":"string //商品简介",
		"sales":"int //销量",
		"shopId":"int //商铺编号",
		"className":"string //类别",
		"goodsPictures":[{
			"id":"int //主键，图片编号",
			"goodsId":"int //商品编号",
			"address":"string //图片地址",
			"status":"int //图片状态，1为默认地址，0为备用地址"
		}],
		"picture":"string //商品的默认图片",
		"num":"int //商品的数量",
		"collected":"int //商品收藏状态，1为已收藏，0为未收藏"
	}
}

返回结果

{
	"code":"string //响应代码，400为错误，200为正确",
	"msg":"string //响应提示信息,若为搜索请求,则为页码总数",
	"data":{}
}

获取申请单列表

请求参数 application/x-www-form-urlencoded

参数名 类型 必须 描述
pageNum int 是 页码
pageSize int 否 单页大小
buyerId string 否 卖家用户编号
sellerId string 否 卖家用户编号
status string 否 申请单状态
